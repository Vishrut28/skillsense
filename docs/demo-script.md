# SkillSense — Demo Video Script (3–5 min)

## Setup before recording
- Open SkillSense in Chrome (localhost:5173 or deployed URL)
- Have Loom or OBS running
- Keep microphone on — you'll use voice input during the demo
- Load sample data is already in the app — no typing needed

---

## Script

### [0:00–0:30] Hook
"A résumé tells you what someone claims to know. This is SkillSense — it actually finds out."
→ Show the landing page. Point to the particle background, the tagline.

### [0:30–1:00] Input
"I'll paste in a Senior Frontend Engineer job description and a candidate résumé. 
Or I can just hit 'Load sample' for both."
→ Click both Load Sample buttons. Show the JD (left) and résumé (right).
"Hit Start Assessment."

### [1:00–1:45] Briefing + first question
→ Watch the extraction spinner ("Analysing job requirements...")
→ Briefing screen appears: "Aria introduces herself. 7 skills identified. TypeScript, React, WebSockets..."
"Aria is our AI interviewer — she's read the JD and the résumé. She knows Priya claims to be 
'proficient' in React and 'familiar' with TypeScript. Now she'll test that."
→ Click Begin Assessment.
→ First question streams in.

### [1:45–2:45] Live assessment — show voice input
"I'll answer using voice."
→ Click the mic button. Speak an answer (something with "I think" or "maybe" in it).
→ Show the confidence meter lighting up amber ("Moderate — probing deeper").
"The app detected uncertainty in that answer. Watch what Aria asks next."
→ Submit. Second question appears — noticeably probing the same gap.
"She went deeper on exactly what I was uncertain about. That's the key intelligence here."
→ Answer question 2. Type this one. Submit.
→ Answer question 3. Submit.
→ Scoring spinner: "Evaluating your React proficiency..."

### [2:45–3:30] Show live scoring panel + radar
→ Score appears in left sidebar and right panel. Radar chart updates.
"React scored 6 out of 10 — intermediate. Watch the radar chart fill in."
→ Quickly skip through 1–2 more skills (you can type short answers to speed through)
"The radar is building in real time as each skill completes."

### [3:30–4:15] Report walkthrough
→ Final report appears.
"54% overall readiness. Hiring recommendation: hire with plan. Time to job-ready: 8–10 weeks."
→ Click expand on TypeScript score — show the rationale, strengths, critical gaps.
→ Click Learning Plan tab.
"Priority 1: TypeScript. Current score 3, target 8. 6 weeks, 8 hours per week."
→ Show the week-by-week learning path cards.
→ Click a resource. "Real URLs — TypeScript Deep Dive, Matt Pocock's tutorials."
→ Click Weekly Schedule tab. "AM/PM breakdown."
→ Click 30-Day Goals. "Specific, verifiable milestones — not vague advice."

### [4:15–4:30] Close
"This is SkillSense — from résumé to honest assessment to a precise learning plan, 
fully conversational, with voice input, in under 15 minutes. Built on Anthropic Claude 
with a free Google Gemini fallback. Thank you."

---

## Recording tips
- Use Loom: loom.com (free, one click to record + auto-upload)
- 1080p, microphone on
- Don't rush — let the loading states play out, they look good
- If a question takes too long, just narrate while waiting
- Keep answers short (2–3 sentences) to keep pace fast
